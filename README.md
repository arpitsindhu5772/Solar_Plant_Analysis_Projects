## 🚀 Overview
This project focuses on end-to-end analysis of solar power plant performance using real-world SCADA data. It covers plant-level KPIs, inverter-level efficiency, string performance, and fault diagnostics to improve reliability and maximize energy generation.

---

## 📊 Key Analysis Modules

### 📈 Plant Performance Analysis
- PR (Performance Ratio), CUF, Specific Yield
- Monthly & daily generation tracking
- Plant-level efficiency monitoring

### ⚡ Inverter Performance Analysis
- Inverter-wise generation and efficiency
- Availability tracking
- Identification of underperforming inverters

### 🔎 String-Level Analysis
- String-wise comparison
- Detection of faulty/low-performing strings

### 🌤️ Irradiance vs Generation Benchmarking
- Expected vs actual generation comparison
- Loss identification based on irradiance
- Performance deviation analysis

### ⛔ Loss & Curtailment Analysis
- Grid downtime impact
- Curtailment loss calculation
- Soiling and system losses estimation

### 🛠️ Preventive Maintenance (CBM)
- SCADA-based predictive insights
- Early issue detection
- Downtime reduction strategies

### 🖥️ SCADA Data Analysis
- Data cleaning and preprocessing
- Parameter analysis (Voltage, Current, Temperature, Irradiance)
- Trend & anomaly detection

### 🚨 Fault & Alarm Analysis
- Fault classification (Breakdown, Shutdown, Grid Failure)
- Alarm trend analysis
- Root cause identification
